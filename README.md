## Projeto de Teste para Entrega de Prova
Este é um projeto criado para atender aos requisitos da prova, com o objetivo de demonstrar as habilidades em automação de testes e seguir as melhores práticas. O projeto utiliza o padrão Page Object para estruturar e organizar o código.

### Requisitos
Antes de executar o projeto, certifique-se de ter as seguintes dependências instaladas:

- Maven
- JDK (Java Development Kit)

### Configuração do Projeto
Siga as etapas abaixo para configurar o projeto em sua máquina local:

- Clone este repositório para o seu ambiente local ou faça o download do arquivo compactado.
```
bash git clone https://github.com/seu-usuario/nome-do-projeto.git
```
- Navegue até o diretório raiz do projeto.
```
bash cd nome-do-projeto
```
- Compile o projeto utilizando o Maven.
```
bash mvn compile
```

### Executando os Testes
Para executar os testes, utilize o seguinte comando:

```
bash mvn test
```
Os testes serão executados e o relatório de resultados será exibido no console.
